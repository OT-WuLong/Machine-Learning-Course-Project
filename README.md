# 📊 学生成绩预测系统

基于机器学习的学生成绩预测与学情预警系统。输入学生的线上/线下成绩，实时预测总成绩和等级，并提供低分预警和教学干预建议。

---

## 功能特性

- **🎯 个体预测**：自由勾选特征，实时预测总成绩（回归）和成绩等级（分类）
- **📈 总体分析**：特征相关性热力图、渐进预测效果对比、数据统计
- **🔍 模型可视化**：决策树结构图、混淆矩阵、回归系数分析
- **🚨 智能预警**：双模型一致判差时红色预警，分歧时提示参考
- **💡 教学建议**：根据学生薄弱特征自动生成个性化改进建议

## 安装与运行

```bash
# 1. 克隆仓库
git clone <repo-url>
cd 机器学习课设

# 2. 安装依赖
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

# 3. 启动应用
streamlit run app.py
```

浏览器打开 `http://localhost:8501` 即可使用。

## 数据说明

原始数据为某课程 7 个班级共 512 条学生成绩记录，包含：

| 分组 | 字段 |
|:----|:----|
| 线上成绩(60%) | 线上平时成绩、线上章测试、线上期末考 |
| 线下成绩(40%) | 线下互动、线下期末考 |
| 总成绩 | 总成绩(线上+线下) |

## 模型性能

| 模型 | 指标 | 值 |
|:----|:----|:--:|
| 多元线性回归 | R² | **0.9358** |
| 多元线性回归 | MAE | **1.08** |
| 决策树分类 | 准确率 | **85.4%** |

## 使用示例

1. 进入 **个体预测** 页面
2. 勾选你要使用的特征（至少 2 个）
3. 调整滑块或输入数值
4. 选择班级，点击「开始预测」
5. 查看预测结果、预警提示和教学建议
